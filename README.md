# MTF World Cup Guess

A static GitHub Pages app for a World Cup friends prediction game.

## Included database

- 48 World Cup teams with group, code, flag and common aliases.
- Player suggestion list for Golden Boot, Golden Ball, assists and goalkeeper picks.
- Player fields also allow manual typing, so the game is not limited to the suggestion list.

## Features

- Submit friend picks
- Edit/delete participants
- Champion, runner-up, 3rd place, Golden Boot, Golden Ball, assists, goalkeeper, top scoring team and surprise team
- Enter actual results after the World Cup
- Automatic leaderboard
- Editable scoring points
- Lock/unlock picks
- Export CSV
- Export/import JSON backup

## GitHub Pages

Upload all files to the root of your repository, then enable GitHub Pages from the `main` branch and `/root` folder.

Important: this is a browser-only app. It stores data in localStorage, so one admin should keep the master copy, or friends should export JSON and send it to the admin.
