How to use

Recommended flow

  1. Share the GitHub Pages link with your friends.
  2. Each friend submits picks before the deadline.
  3. The admin exports/backs up data, then locks submissions.
  4. After the World Cup, enter the actual winners in Results & Scoring.
  5. Highest score wins. Use CSV export for WhatsApp sharing.
This is a browser-only static app. Data is saved in the device/browser localStorage, not a shared online server. For a group game, one admin should maintain the master copy, or friends should export JSON and send it to the admin.
